A collection of networking labs built using Cisco Packet Tracer

##  Labs Included

###  Routing
- Static Routing : manually configured routes between routers
- Dynamic Routing (RIP) : automatic route learning using RIP protocol 

###  NAT (Network Address Translation)
- Static NAT : one-to-one mapping of private to public IP addresses
- Dynamic NAT : pool-based translation for multiple internal hosts

###  Subnetting
- VLSM (Variable Length Subnet Mask) : different subnet sizes (/25, /26, /27, /30) within the same network topology

##  Requirements
- Cisco Packet Tracer — free via Cisco NetAcad
- Download at: netacad.com

##  Files
- Each folder contains a `.pkt` file for that lab
- Open directly in Packet Tracer to view and interact with the topology
